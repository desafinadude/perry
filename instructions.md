Write a REACT webapp that allows playing midi with connected keyboard through the web midi api. The app should be able to load soundfonts and have the ability to split the keyboard into different zones, each zone can be assigned a different soundfont. The app should also have a visual representation of the keyboard and the ability to record and playback midi sequences.
There is already a font.sf2 file in the folder with multiple channels that can be used for the soundfonts. The app should allow the user to select which channels to use for each zone and also allow the user to adjust the volume and other parameters for each zone.
